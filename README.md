# Stock-Service 
Le **Stock-Service** est un micro-service permettant de gérer les cotations boursières des entreprises.  
Il fournit des opérations CRUD (Créer, Lire, Mettre à jour, Supprimer) sur les cotations et peut être interrogé par des clients ou agents AI via MCP.

---

## Fonctionnalités
- Ajouter une cotation pour une entreprise.
- Consulter la liste des cotations.
- Mettre à jour une cotation existante.
- Supprimer une cotation.
